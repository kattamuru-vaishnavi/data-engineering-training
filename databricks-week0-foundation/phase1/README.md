# Databricks Week 0 - Foundation

## Phase 1: SQL to PySpark – Simple Foundation

### Overview
In this phase, I worked on building a basic understanding of how SQL queries can be translated into PySpark DataFrame operations.

I practiced fundamental operations like filtering, selecting columns, and grouping data using both SQL and PySpark.

---

###  What I Implemented
- Show all customers  
- Filter customers by city (Chennai)  
- Filter customers with age > 25  
- Select specific columns (customer_name, city)  
- Count customers city-wise  

---

### SQL to PySpark Mapping

| SQL | PySpark |
|-----|--------|
| SELECT * | show() |
| WHERE | filter() |
| SELECT columns | select() |
| GROUP BY | groupBy() |
